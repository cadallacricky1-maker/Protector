# Privacy Policy for Protector App

**Last Updated: December 14, 2025**

## Introduction

cadallacricky1-maker ("we", "us", or "our") operates the Protector mobile application (the "Service"). This page informs you of our policies regarding the collection, use, and disclosure of personal data when you use our Service.

## Information We Collect

### Information You Provide
- **Voice Patterns**: When you enable voice authentication, we store voice pattern data locally on your device in encrypted format
- **Location Preferences**: Geo-fence locations and proximity radius settings
- **Account Information**: Email address for subscription management (via Google Play)

### Automatically Collected Information
- **Location Data**: Real-time location when protection is active (stored locally, not transmitted)
- **Device Motion**: Accelerometer data for theft detection (processed locally, not stored)
- **Usage Analytics**: App usage patterns, feature usage, crash reports (anonymous)

### Premium Subscription Data
- **Subscription Status**: Managed by Google Play Billing
- **Purchase History**: Stored by Google Play Store
- **Payment Information**: Handled entirely by Google Play (we never see payment details)

## How We Use Your Information

### Core Functionality
- **Theft Detection**: Accelerometer data is processed in real-time to detect suspicious movement
- **Proximity Alerts**: Location data triggers warnings when device moves beyond set radius
- **Voice Authentication**: Voice patterns verify authorized users

### Premium Features (Paid Subscribers Only)
- **Cloud Backup**: Voice patterns and settings backed up to secure cloud storage (encrypted)
- **Multi-Device Sync**: Settings synchronized across your devices
- **Advanced Analytics**: Historical data for insights (365 days)

### Service Improvement
- **Crash Reports**: Anonymous crash data helps us fix bugs
- **Usage Analytics**: Understand which features are used to improve the app
- **A/B Testing**: Test new features with subset of users

## Data Storage and Security

### Local Storage
- **Voice Patterns**: Encrypted using Android Keystore
- **Location History**: Stored locally for 7 days (free) or 365 days (Premium)
- **Preferences**: Stored in encrypted SharedPreferences

### Cloud Storage (Premium Only)
- **Encryption**: All cloud data encrypted in transit (TLS 1.3) and at rest (AES-256)
- **Access Control**: Only you can access your data with your Google account
- **Data Centers**: Stored on secure Google Cloud Platform servers

### Security Measures
- Code obfuscation with ProGuard/R8
- Certificate pinning for network requests
- Secure key storage using Android Keystore
- Regular security audits
